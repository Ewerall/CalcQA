Обе программы реализованы как класс ModernCalculator на основе Tkinter. Основные компоненты:
  Графический интерфейс с дисплеем и кнопками
  Обработчики событий для кнопок
  Валидация ввода
  Логика вычислений
  Обработка ошибок

Ключевые методы:
__init__() - инициализация интерфейса:
  Создание окна с фиксированным размером
  Настройка шрифтов
  Создание дисплея (Entry widget)
  Создание кнопок с цветовой дифференциацией
  Привязка hover-эффектов

set_calculator_icon() - попытка загрузки иконки приложения (имеет проблемы в обеих версиях)
lighten_color() - осветление цвета кнопки при наведении курсора
is_valid_expression() - валидация математического выражения 
is_valid_next_char() - проверка допустимости следующего символа 
click_handler() - обработчик нажатий кнопок

Принцип работы:
Пользователь вводит выражение с помощью кнопок
  При нажатии "=":
    Выражение валидируется
    Спецсимволы заменяются на математические (×→*, ÷→/)
    Выполняется вычисление через eval()
    Результат выводится на дисплей
  Обработка особых кнопок:
    C - очистка дисплея
    ⌫ - удаление последнего символа
